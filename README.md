# lepto-webp

```console
$ npm i -D lepto-webp
```

See [lepto documentation](https://github.com/dimitrinicolas/lepto) for usage.

This plugin has no options, just add it to your plugins list.

Default options:
```js
"use": [
  {
    "name": "lepto-webp",
    "quality": 80,
    "alphaQuality": 100,
    "lossless": false,
    "replaceFile": false
  }
]
```
